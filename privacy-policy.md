FocusTube Privacy Policy

FocusTube does not collect, store, or transmit any personal user data.

All extension settings are stored locally in the user's browser using Chrome storage.

The extension only modifies the YouTube interface to remove distracting elements such as Shorts, recommendations, and comments.

No browsing history, personal information, or account data is collected or transmitted to external servers.

FocusTube does not sell, transfer, or share user data with third parties.

If you have questions about this privacy policy, please contact the developer via the GitHub repository.
